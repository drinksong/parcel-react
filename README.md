# Getting started

1. install width yarn

    ```
    yarn install
    ```

    or with npm

    ```
    npm install
    ```

2. run in dev environment

    ```
    yarn start
    
    # or
    
    npm start
    ```
    
3. production

    ```
    yarn run build
    
    # or
    
    npm run build
    ```
    
# Environment

Based `babel 7`, not support node versions includes `0.10, 0.12, 4, 5`
